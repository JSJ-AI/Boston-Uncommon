Welcome to Boston UnCommon

Agentic AI · Computer Vision · 911 Deployment Simulation

An agentic AI computer vision actionable intelligence model that uses computer vision and object detection from 911-call audio attributes — speech-to-text that spawns object search or pursuit.

CV AI models leverage in-camera, edge-server, and cloud processing coordinated with traffic cameras, surveillance cameras, drones, dash cams in police cars, and body-worn cameras for officers on bike or on foot.

Press “Answer 911 Call” to put the Security Operations Center into action.

An event object or person will be placed on the map based on call information, and then all the cameras’ fields of view can be used to locate and vector resources to resolve the 911 call tasking.

This is a thought-stimulation exercise to imagine what is possible when computer vision telemetry data can be coordinated and tasking to law enforcement help is automatically deployed.
